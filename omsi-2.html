OMSI 2

  Get OMSI 2 up and running

OMSI 2 is a DLC. Simply purchase OMSI 2 from Stream and you're good to go!

System Requirements

Operating System: Windows 7 or later
Processor: Pentium 4 @ 2.6GHz or better
Memory: 2 GB RAM
Graphics: GeForce or comparable AMD with minimum 512 MB
DirectX: Version 9.0c

Note: You may require a better processor, memory and graphics card to run the game with better performance.



Step 1 - Purchase the Program from Steam



Step 2 - Familiarizing the OMSI 2 Directory

Only attempt the following if you are familiar with Modifying OMSI contents, and have backed up your item. We will not assume responsiblity should things go wrong.

Our released buses are a complete package. Hence you would only need to download the following :
- Fonts (for BOTH Bus & Maps)
- Sceneryobjects (for Maps)
- Splines (for Maps)
- Krueger EDS Scripts (for Bus)

Step 1 - Downloading our Maps/Buses

NOTE: This is applicable for all released buses. For this guide, we'll be demonstrating how to do so for the Solaris Urbino 18.
First up, download the Bus.

Step 2 - Extracting it into your OMSI 2 Directory

Open the Fonts & Bus Zip file and extract the folder into the OMSI directory (C:\(your Directory)\steamapps\common\OMSI 2).

We recommend using 7-Zip or NanaZip (shown in the video) to extract.
