Yishun LRT

Developer(s)

willis_chong   SMRTO Studios

Description

The Yishun LRT Line is a fictional line mainly based on Singapore. The route is made by willis_chong. Upgrades are done in-house by SMRTO Studios.

Recommended OpenBVE Version: 1.9.27 (64-bit)
