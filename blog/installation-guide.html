OpenBVE Releases Installation Guide (Trains)

We have previously released 4 new trains, and in this blog, we'll guide you through the process of downloading and installing the trains.
Links:
- Alstom R151
- KSF C151C
- KSF C151B
- KSF C151A

1. Downloading our Trains

NOTE: This is applicable for the Alstom R151, KSF C151A/B/C trains only. For this tutorial, we'll be demonstrating how to do so for the R151.

First up, download the Base Train. For the R151, the Base Train can be downloaded here.

Once downloaded, open the folder. Select 'SMRT Alstom MOVIA R151 (825-826) (825F)' and extract it into your Train Folder (i.e. the directory where your OpenBVE trains are saved.)

2. Overwriting our modifications into the Base Train

Now that you've extracted the Base Train's folder, it's time to put our modifications in!

We recommend using 7-Zip or NanaZip (we currently use this).



Open up the Modification zip file and extract it into your Train Folder (make sure its at the same directory where you stored your Base Trains earlier.)

If you've done the steps correctly, the 7-Zip/NanaZip will prompt you to overwrite the Base Train's content. Select 'Yes to all'.

You are now ready to use the train. These steps apply for the KSFs too! Do note that we recommend OpenBVE Version 1.9.27 (64-bit) or higher for optimal experience.

Note: This guide is for the installation on Windows 11 only. If you are using other versions of Windows, some steps may differ from the list below. If you are on another operating system such as Mac OS, you may wish to ignore this guide.
